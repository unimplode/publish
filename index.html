Bio

Hi.

I create dark, weird, hopeful, psychedelic, obsessive 2D animation, and robust, minimalist, user-centric code. I work in 2D animation, wearing all the hats from writer to director to animator and love to collaborate.

Currently I divide my time between 2D animation contracts, teaching programming and Unity, and continually pushing forward the code and design my immersive VR Animation tool, Arpeggio.

If you're interested in any aspect of what I do, just write! I'm always game for a coversation.
